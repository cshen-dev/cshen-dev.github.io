# Charlie Shen's Happy Stack


## Prerequisite 

- macOS
- Linux terminal knowledge


## Usage

**Install brew**

[Homebrew â€” The Missing Package Manager for macOS (or Linux)](https://brew.sh/)

**Install Ruby via rbenv**
```
brew install rbenv
rbenv install
rbenv init (if you haven't run once)
```

**Install Ruby dependencies**
```
bundle
```

**Run Jekyll locally**

```
./tools/run.sh
```

**Test site contents**

```
./tools/test.sh
```
